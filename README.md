# Install

    $ cd ~
    $ git clone git@github.com:qcl/qcl.rc.git .qclrc
    $ cd .qclrc
    $ .copyConf.sh



## Zash

* Install [Oh My zsh](https://github.com/ohmyzsh/ohmyzsh) first

* Copy `zshrc` to `~/.zshrc`  

* May need to change path of `export ZSH=""` in `.zshrc` file.
